## Featured Projects

### Industrial 6D Pose Estimation System &nbsp;·&nbsp; [View →](https://github.com/Sruthi-Reddy-B/Industrial-6D-Pose-Estimation-System)

Full pipeline for estimating the 6D pose (position + orientation) of industrial objects from monocular camera input — built from my master's thesis work at BEC GmbH / RWTH Aachen.

- YOLOv5 trained on synthetically generated Blender data with domain randomisation
- PnP solver for recovering rotation and translation from 2D–3D correspondences
- Pose overlay visualised on live camera frames in real time

`Python` `PyTorch` `OpenCV` `YOLOv5` `Blender` `Synthetic Data` `PnP Solver`
